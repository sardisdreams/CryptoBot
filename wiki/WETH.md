# Wrapped Ether (WETH)

## What it is
WETH is ETH wrapped in an ERC-20 contract so it can be used in DeFi swaps. On Base, ETH and WETH are interchangeable — the bot holds native ETH and wraps it automatically when swapping.

## Fundamentals
- Ethereum is the largest smart contract platform by TVL and developer activity
- Base is built on Ethereum's security via the OP Stack
- ETH is used for gas on Ethereum mainnet; on Base, ETH is also the gas token
- Supply: ~120M ETH, deflationary post-merge (EIP-1559 burns)
- Staking yield: ~3-4% APY (via validators)
